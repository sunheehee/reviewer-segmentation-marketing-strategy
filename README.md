# 🛒 리뷰 데이터를 활용한 쿠팡·테무 고객 세분화 및 플랫폼 전략 제안

## 프로젝트 개요
본 프로젝트는 쿠팡과 테무의 **리뷰 데이터를 기반**으로,  
고객의 **리뷰 행동과 감성 특성에 따라 고객 유형을 세분화**하고  
**플랫폼 운영 전략을 도출**하기 위한 데이터 분석 프로젝트입니다.

리뷰 텍스트 감성 분석 결과와 리뷰 행동 지표를 결합하여 군집분석을 수행하고,  
통계적 검정을 통해 세그먼트 간 차이를 검증함으로써  
분석 결과를 전략 제안으로 연결하는 데 목적을 두었습니다.

---

## 👨‍👩‍👧‍👦 팀 구성 및 역할
- 팀 프로젝트 (5인)
- 나의 역할
  - 테무 리뷰 데이터 수집 (UI Path 기반 웹 스크래핑, 팀원과 공동 수행)
  - 고객 세그먼트 해석 및 플랫폼 전략 제안 (단독 수행)
- 전체 프로젝트 기여도: 약 55%

---

## 📦 데이터 개요
- 쿠팡: 상품 리뷰 웹 크롤링
- 테무: UI Path 기반 리뷰 수집
- 분석 단위: 리뷰 작성 고객
- 규모: 플랫폼별 약 1,000명
- 데이터 구성
  - 리뷰 텍스트
  - 별점
  - 리뷰 작성 빈도 및 최신성
  - 리뷰 길이, 공백 리뷰 여부 등

※ 데이터 원본은 공개하지 않았으며, 코드만 포함되어 있습니다.

---

## 🎯 분석 방향
1. 리뷰 텍스트 전처리 및 감성 분석 (KoBERT 기반)
2. 리뷰 행동 및 감성 지표 기반 파생 변수 생성
3. PCA 기반 차원 축소 후 K-Means 군집분석
4. ANOVA 및 Tukey HSD를 활용한 군집 간 차이 검증
5. 고객 세그먼트별 특성 해석 및 플랫폼 전략 도출

---

## 💡 주요 결과 및 인사이트
- 플랫폼별 고객 세그먼트 도출
  - 쿠팡: 5개 클러스터
  - 테무: 4개 클러스터
- 통계 검정을 통해 군집 간 주요 변수에서 유의미한 차이 확인
- 리뷰 행동·감성 특성을 반영한 고객 유형 정의
